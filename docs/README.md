# Documentation

## Structure

- **plans/** - Project plans, tasks, architecture decisions, and roadmaps
  - **main.md** - Main work tracking document (start here!)
- **references/** - Documentation, research notes, API references, and learning resources  
- **agents.md** - AI agent configuration (source of truth for AGENTS.md, CLAUDE.md, GEMINI.md, .github/instructions.md)

## Temporary Files

Use `tmp/` for scratch work, logs, and temporary outputs. This directory is gitignored.
